## 📋 About the CISA Exam

The CISA certification, issued by **ISACA**, is a globally recognized credential for IT audit, control, and security professionals. The exam covers 5 domains:

1. **Domain 1** – Information Systems Auditing Process
2. **Domain 2** – Governance and Management of IT
3. **Domain 3** – Information Systems Acquisition, Development, and Implementation
4. **Domain 4** – Information Systems Operations and Business Resilience
5. **Domain 5** – Protection of Information Assets

---

## 📦 File Format

All questions are in **GIFT (General Import Format Technology)** format — a plain-text format widely used for quiz creation.

### Example GIFT Question
```
::Q1:: Which of the following BEST describes the primary objective of an IS audit?
{
  =To provide assurance that IT systems support business objectives
  ~To detect and prevent all IT fraud
  ~To replace the role of IT management
  ~To ensure compliance with programming standards
}
```

## 🚀 How to Use

### Import into Moodle
1. Go to your Moodle course → **Question Bank**
2. Click **Import**
3. Select **GIFT format**
4. Upload the desired `.gift` file
5. Click **Import** to confirm

### Import into Other LMS
Most LMS platforms that support GIFT format follow a similar import workflow. Refer to your platform's documentation for specific steps.
